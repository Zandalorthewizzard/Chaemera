# Documentation Synchronization Procedure

After substantial changes:

1. Update `INDEX.md` navigation.
2. Update `inventory.json` records.
3. Review `notes/` for durable findings that should be promoted into canonical docs.
4. Append `progress.md` journal entry.
5. Update claims/evidence in `validation.md`.
