# AGENTS Documentation Rules Template

1. Code is source of truth.
2. Canonical docs require evidence blocks.
3. Unknowns must be marked `UNKNOWN` or `deferred`.
4. Sync index, inventory, progress, validation after substantial changes.
5. Keep dated working notes in `notes/` when continuity, debugging, or handoff context matters.
6. Treat `notes/` as non-canonical; promote durable findings into evidence-backed docs.
